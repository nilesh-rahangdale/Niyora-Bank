# Niyora Bank - Core Banking Management Portal

A modern enterprise core banking management system frontend built with React, Redux Toolkit, and Tailwind CSS.  
Designed with a premium dark-mode glassmorphic UI to simulate real-world banking operations with role-based workflows.

---

##  Features & Role-Based Modules

###  Admin Module
- Manage branches and banking staff
- Create users (Manager, CSO, Teller)
- View staff details and manage credentials
- Secure password reset operations

###  Manager Module
- Branch performance dashboard with analytics
- Monitor teller cash drawers and balances
- Review and resolve customer complaints
- Track operational activities

###  CSO Module
- Customer onboarding with KYC verification
- Manage customer documents (Aadhaar, PAN, Passport, etc.)
- Approve/update KYC details
- Create Savings and Current accounts

###  Teller Module
- Handle banking transactions:
  - Cash deposit
  - Withdrawal
  - Fund transfer
  - Cheque processing
- Manage cash drawer operations
- Perform end-of-day reconciliation

---

##  Tech Stack

- **Frontend:** React 18, React Router v6
- **State Management:** Redux Toolkit
- **Styling:** Tailwind CSS v4
